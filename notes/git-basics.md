# Git Basics

## Что такое Git
Git — это система контроля версий.

## Базовые команды
- `git init` — создать репозиторий
- `git add .` — добавить изменения в индекс
- `git commit -m "message"` — зафиксировать изменения
- `git status` — показать текущее состояние
- `git log` — показать историю коммитов

## Работа с ветками

- git branch — список веток
- git checkout -b <name> — создать ветку
- git merge — объединить ветки
