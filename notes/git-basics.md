# Git Basics

## Что я делал

Во время изучения Git я:

- создал локальный репозиторий
- подключил GitHub
- сделал первый commit
- пушил изменения на GitHub
- проверял статус проекта
- изучал историю commit
- создавал новые ветки
- переключался между ветками
- работал через feature branch
- делал merge веток
- обновлял README
- добавлял notes файлы
- работал полностью через консоль

---

## Что я понял

Git помогает:
- сохранять историю проекта
- не терять изменения
- безопасно экспериментировать
- работать с несколькими версиями кода

Ветки позволяют делать изменения отдельно от основной версии проекта.

GitHub можно использовать как:
- портфолио
- хранилище проектов
- платформу для командной работы
