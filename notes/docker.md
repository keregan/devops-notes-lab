# Docker

## Что я делал

В рамках изучения Docker я:

- установил Docker
- проверил работу Docker через консоль
- запустил первый контейнер nginx
- изучил разницу между image и container
- посмотрел список контейнеров через `docker ps`
- останавливал и запускал контейнеры
- удалял контейнеры
- создал собственный Dockerfile
- собрал свой image через `docker build`
- запускал контейнер из собственного image
- создал docker-compose.yml
- запускал сервис через `docker compose up`
- использовал флаг `--build`
- подключался внутрь контейнера через `docker exec -it`
- запускал команды внутри контейнера
- изучал работу портов
- подключил .env файл
- добавил environment variables
- добавил Redis контейнер

- изучал volumes
- настраивал сохранение данных контейнера

---

## Что я понял

Docker помогает запускать приложения в одинаковом окружении.

Контейнеры позволяют:
- быстро запускать сервисы
- изолировать приложения
- переносить проекты между устройствами
- упрощать деплой

Docker Compose упрощает запуск нескольких сервисов одновременно.
