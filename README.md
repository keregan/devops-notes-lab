# DevOps Notes Lab

Практический репозиторий для изучения DevOps по маленьким шагам.

## Цели
- закрепить Git, Linux, Docker, CI/CD
- собирать полезные команды и заметки
- делать маленькие практические задания
- постепенно превратить репозиторий в портфельный проект

## Структура
- `notes/` — краткие заметки по темам
- `practice/` — практические команды и мини-задания
